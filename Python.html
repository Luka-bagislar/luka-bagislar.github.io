Ker naša spletna stran govori o programiranju v programskem jeziku Python, smo vam spodaj zapisali nekaj zanimivih podatkov o programu python.

Uporaba Pyhtona

Python se redno uporablja za razvoj spletnih strani in programske opreme, analizo podatkov, vizualizacijo podatkov in avtomatizacijo opravil. Uporabljajo ga velika podjetja, kot so naprimer Nasa, Pixar, Netflix, Facebook, Spotify in druge. Je eden od štirih glavnih jezikov Googla, medtem pa je tudi googlov youtube skoraj v celo zapisan v Pythonu. Zaradi svoje enostavne uporabe je priljubljen tudi pri »neprogramerskih« poklicih, kot so računovodje, znanstveniki itd. Te uporabljajo python predvsem za analizo podatkov in avtomatizacijo opravil.

Zgodovina

Python je programski jezik, katerega avtor je Guido van Rossum. je 20. Februarja leta 1991. Ena od zanimivosti tega jezika je to, da je jezik »one persons work«, kar pomeni da ima enega avtorja. Običajno so avtorji programskih jezikov velika podjetja, za katere dela veliko programerjev. Seveda vseh komponent/funkcij v programu Python ni ustvaril Guido sam, vendar tista prva ideja kako bi Python deloval in izgledal, prišla iz njegove glave.

Zakaj je Python najboljši program za začetnike?

Veliko strokovnjakov pravi, da je Python odličen jezik za začetnike, saj so njegove kode dobro berljive. Python omogoča neskončno možnosti za uporabo kod in učenje iz že prikazanih primerov. Poleg tega je čas, ki ga potrebujemo vložiti v učenje programa Python, znatno krajši kot za druge programske jezike. Kar pomeni, da lahko hitreje začnemo programirati v Pythonu. Program je tudi »Free to use«, kar pomeni, da je njegova uporaba povsem brezplačna. Poleg tega pa lahko python uporabljamo na vseh sodobnih programskih sistemih.
